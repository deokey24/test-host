### 클래스 소개

논술 뿐만 아니라 수능 국어영역, LEET 언어이해 등 학술적 텍스트에 기반한 시험 모두에서 적용할 수 있는 독해방법론입니다.

**핵심 내용:**
- 연결독해는 문장 하나하나를 단발적으로 이해하는 것이 아니라 전체 글의 내용이 하나로 이어지고 있다는 전제 속에서 이루어지는 분석법
- 모든 글이 표현은 다르지만 결국 같은 의미로 통용된다는 점을 체화하는 수업
- 현행 논술시험은 지문의 난도를 높여 지문 이해도를 중점으로 평가하는 경향
- 지문에 대한 깊이 있는 이해와 독해 및 분석력 향상이 중요
- 논술뿐 아니라 다양한 학술 텍스트 독해에 활용

**추천 대상:**
- 연고대 편입 준비생

- 수능 국어영역 준비생

## 8. 기술 스택 및 구조

### 기술 구성

| 항목 | 내용 |
|------|------|
| 구조 | 단일 HTML 파일 SPA (Single Page Application) |

### JavaScript 주요 함수

| 함수 | 역할 |
|------|------|

| `filterTab(el, type)` | 홈 클래스 필터 탭 전환 |
| `cpFilter(el)` | 클래스 페이지 필터 탭 전환 |

| `initReveal()` | IntersectionObserver 기반 스크롤 reveal 초기화 |
| `toggleMobileMenu()` | 햄버거 메뉴 열기/닫기 |
| `closeMobileMenu()` | 모바일 메뉴 닫기 |
